第一种字符串格式化

语法

“%占位符 %占位符” % （变量1,变量2)

常用占位符
占位符 数据类型
%s 字符串
%d 整数
%f 浮点数
精度控制语法

m控制数字宽度，.n控制小数点位数

m和.n均可以省略

如果m比数字本身宽度还小，则m不生效

.n会对小数部分做精度限制，同时也会做四舍五入

  • 如%5.2f，%5d，%.2f等

第二种字符串格式化

语法

f” {变量} {变量}”

  • 不理会类型

  • 不做精度控制

适合对精度没有要求的时候快速使用
